This guide covers common issues and their solutions for Bazytrack Go, developed based on the Traccar platform.



üöó Device & Tracking Issues

  1. Device is not showing online


    • Check if the SIM card is active and has sufficient data.

    • Ensure the device is powered on and connected correctly.

    • Verify the device is configured with the correct server IP and port.

    • Confirm the correct APN settings for the network provider.

    • Check server firewall settings and port forwarding if using a custom server.


  2. No location updates / Position not updating


    • Ensure the device is in an area with clear GPS visibility.

    • Check whether the device requires ignition to send updates.

    • Confirm the last update time in the device list.

    • Inspect server logs to see if data is being received.


  3. Incorrect location or bouncing positions


    • Update device firmware if needed.

    • Ensure the GPS antenna is placed away from metal or interference.

    • Adjust Traccar position filtering settings (e.g., validity, accuracy).


  4. Device showing wrong time


    • Verify timezone settings on the device and in Traccar.

    • Confirm the Traccar server and database use UTC.

    • Ensure device firmware synchronizes time correctly.


  5. Speed not showing or incorrect


    • Check if the device supports and sends speed data.

    • Confirm protocol compatibility with Traccar.



      üîî Alerts & Notifications

  6. Alerts not triggering


    • Ensure the alert type is supported by the specific device.

    • Check if the alert is configured on the correct level (device or group).

    • Make sure the triggering condition is being met.

    • Review if the notification delivery method (email, SMS, etc.) is set up correctly.


  7. Notifications not received


    • Verify email or SMS settings in the user account.

    • Check spam/junk folders.

    • Ensure SMTP server configuration is valid if using custom email settings.



      Reports & History


  8. Reports are empty or incomplete


    • Check that there is tracking data for the selected time range.

    • Ensure the correct device is selected.

    • Verify the time filter (UTC vs local time).


  9. History playback not working


    • Confirm there is valid position data for the selected period.

    • Check filtering settings in the UI (e.g., show only valid positions).



      üöÄ Performance & System

  10. System is slow or not loading


    • Restart the Traccar service.

    • Check system resource usage (CPU, RAM).

    • Clean up old logs or unnecessary historical data.

    • Ensure server disk is not full.


  11. Cannot login


    • Verify correct username and password.

    • Check if the user account is blocked.

    • Reset password using admin panel or database.


  12. Map is not loading

    • Check internet connection.

    • Verify map provider settings.

    • Inspect browser console for errors.



      üîê User & Permissions

  13. User cannot see their devices


    • Check user-device permissions.

    • Make sure the device is assigned to the user or their group.


  14. Cannot add or edit device


    • Ensure the user has the required permissions.

    • Verify role configuration in Traccar (admin, manager, user).



      üö® Other Common Issues

  15. Fuel level not showing


    • Check if the device supports fuel monitoring.

    • Configure the correct sensor settings in Traccar.


  16. Device shows static location


    • Ensure the device is actually moving.

    • Inspect whether ignition is required to send updates.

    • Review last movement time and speed fields.
